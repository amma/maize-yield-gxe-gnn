# maize-yield-gxe-gnn

Code and supporting materials for the manuscript:

**LSTM-Attention-Guided Graph Neural Networks for Integrated Genotype-Environment Modeling in Maize Yield Prediction**

This repository is being prepared for public release. Scripts and reproducibility materials will be added here selectively.

## Scope

- Training and evaluation scripts for maize yield prediction with genotype-environment modeling.
- Supporting analysis scripts used for manuscript figures and results.
- Data descriptions and links to public datasets where possible.
